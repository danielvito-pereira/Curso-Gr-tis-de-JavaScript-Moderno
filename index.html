Oque faz o JavaScript ?

cliente X server. Exemplo de cliente : computador e celular.
servidores fornecem os dados , usando o navegador como intermediario desviando o cliente para o servidor sendo direcionado pelo navegar. dento do servidor existem pastas, o navegador vai solicitar uma copia do que foi pedido pelo cliente, então o servidor envia uma copia para o navegador que envia para o cliente dentro do navegador que vai traalhar para gerar o visual do site/pagina.
JavaScript é uma tecnologia cliente X site
